# Backlog MVP Brasaland Digital

## Prioridad P0 (inmediata)

- API central con endpoints de salud, locales y resumen de ventas.
- Dashboard ejecutivo basico con ventas y ticket promedio.
- Ingestion inicial de eventos de ventas.
- Workflow de alerta de local abierto sin ventas.
- Workflow de reporte semanal ejecutivo.

## Prioridad P1 (siguiente iteracion)

- Modulo de inventario con riesgo de quiebre por local.
- CRM minimo con historial de pedidos por cliente.
- Catalogo de proveedores con historial de precios.
- Panel operativo para supervisores.

## Prioridad P2 (expansion)

- Recomendador de productos para fidelizacion.
- Busqueda semantica de recetas/procedimientos (RAG).
- Agente de soporte operativo para managers de local.
- Automatizacion de onboarding de cocina de punta a punta.

## Criterio de priorizacion

- Impacto en ingresos o costos.
- Reduccion de carga operativa manual.
- Dependencias tecnicas.
- Tiempo de entrega para demo.
