# Sitio web público de Brasaland

Este directorio contiene la primera versión del sitio web público de Brasaland, cumpliendo los requisitos de accesibilidad, SEO, responsive y validación de formulario, usando Tailwind CSS.

## Estructura

- `index.html`: Landing page con presentación, beneficios, contacto y navegación.
- `formulario.html`: Página de aplicación/registro con validación completa en JavaScript.
- `formulario.js`: Lógica de validación y mensajes de error para el formulario.

## Cómo probar localmente

1. Abre los archivos HTML en tu navegador (no requiere backend).
2. El diseño es responsive y accesible.
3. El formulario valida todos los campos antes de "enviar" y muestra mensajes claros.

## Cumplimiento de requisitos

- HTML5 semántico y etiquetas ARIA.
- Imágenes con `alt` descriptivo.
- SEO con meta y Schema.org.
- Tailwind CSS para todo el diseño.
- Sin CSS personalizado.
- Validación y mensajes de error claros.
- Mobile-first y breakpoints para tablet/escritorio.
